# Roblox Lifting Simulator

Проект симулятора качалки (Lifting Simulator) на платформе Roblox. Игрок кликает, поднимает вес, зарабатывает очки силы и прокачивает своего персонажа.

## 🚀 Структура проекта

Архитектура проекта разделена на логические модули:

*   `DataManager` — отвечает за сохранение и загрузку данных игрока (DataStore).
*   `GameModeManager` — управляет игровыми режимами и общим состоянием сервера.
*   `LiftingHandler` — серверный скрипт, обрабатывающий логику поднятия веса и прокачки.
*   `CharacterScaler` — отвечает за динамическое увеличение размеров персонажа при росте силы.

## 📁 Структура папок в Roblox Studio

*   `StarterPack/SimpleWeight` — стартовый инструмент (гантель) для игрока.
*   `StarterGui/ScreenGui/StrengthLabel` — интерфейс отображения текущей силы на экране.

## 🛠️ Стек технологий

*   **Платформа:** Roblox Studio
*   **Язык программирования:** Luau (Lua)
